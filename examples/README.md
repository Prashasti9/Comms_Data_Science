# Examples

Runnable examples for `simple_viz`. The script builds a small pandas
DataFrame and calls each public function, saving the result as a PDF.

## Run

From the repo root (with the package installed, e.g. `pip install -e .`):

```bash
python examples/generate_examples.py
```

## Outputs

| File | Function | Description |
| --- | --- | --- |
| `line.pdf` | `line_plot` | Monthly revenue over time |
| `bar.pdf` | `bar_plot` | Revenue by month |
| `scatter.pdf` | `scatter_plot` | Spend vs revenue |
| `histogram.pdf` | `histogram` | Distribution of scores |
| `all_plots.pdf` | all four | 2Ă—2 grid using the shared `ax` argument |

The PDFs are regenerated by re-running the script.
